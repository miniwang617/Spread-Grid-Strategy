# Spread-Grid-Strategy
本项目包含三种量化交易策略（价差网格、囤币、做市接针）
# 1. 价差网格策略 (Spread Grid Trading)
- 在价格波动区间内设置多个买卖网格
- 低买高卖，赚取价差收益
- 适合震荡行情

# 3. 做市接针策略 (Market Making)
- 捕捉市场快速上涨或下跌后的反弹机会
- 在极端行情中提供流动性
- 风险较高，需要严格止损
